Different between local storage vs session storage ?

(1) Local storage
The local storage is an object that helps to store data with no expiration date . It is persisted through page relods and stays with the browser until it is cleared manually or trough a setting . The user can access a pages local storage without any time limit . So , it is possible to obtain data the next day or after a month or a year.
(2) Session storage
Session storage acts in a similar way to local storage .But it stores data only for ane session . Therefore the data is deleted when the user close the browser tab .
Finally we can said :
Local storage is a type of web storage object avilavle in HTML 5 that allows storing data with on expiration data . In the other hand , Session storage is type of web storage object avilable in HTML5 that storing data for one session .
